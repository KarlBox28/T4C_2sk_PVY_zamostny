Jaký je rozdíl mezi CSS vlastnostmi „padding“ a „margin“.

Rozdíl je jednak v umístění, padding je přímo okolo "contentu", zatímco margin je až poslední vrstva. Taktéž se margin běžně používá k nadefinování mezer mezi objekty.

Máme element „div“ o velikosti 50px a marginu 20px. Jak zajistíme, aby byl opravdu širkový 50px? Popište vlastními slovy.

Pokud chceme, aby celková šířka byla 50px, musíme nastavit šířku contentu na 10px. Potom se z každé strany nabalí 20px marginu a celková velikost je 50px.

Máme-li nastavený box-sizing na border-box, pak se do šířky - width počítá content, padding a border

BONUS: Uvažujte následující CSS kód. Jaká je celková šířka (width) elementu? Vypište výpočet a odůvodnění. (3 bonusové body) .box { width: 250px; padding: 10px; margin: 5px; box-sizing: content-box; }

Protože máme vlastnost box-sizing nastavenou na content box, platí vlastnost width jenom na samotný tag a ostatní se nabaluje kolem. => šířka = width + padding * 2 + margin * 2 = 250 + 20 10 = 280px
